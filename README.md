# React Challenges

A collection of 20+ mini React projects designed to practice patterns, build competency, and explore React + TypeScript fundamentals.

## Purpose

This repository contains a variety of mini React apps created to strengthen React fundamentals, explore component patterns, and practice using TypeScript and modern front-end libraries.
Each challenge demonstrates a different concept, from state management and event handling to hooks and UI composition.

## Tech Stack

- React
- React Router
- TypeScript
- TailwindCSS for styling
- Tanstack Query for async state management
- Toastify for user notifications

Each mini app demonstrates different React patterns, such as:

- Component composition and props handling
- State management with useState and Tanstack Query
